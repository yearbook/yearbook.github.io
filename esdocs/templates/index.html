Adobe ExtendScript API documentation

Alternative documentation system for Adobe ExtendScript API

We at Yearbook Machine make extensive use of the ExtendScript API on a daily basis, but found the documentation system difficult to deal with. This is our solution to the problem. An attempt at making the ExtendScript API documentation more accessible and easier to use than the native Object Model Viewer.

Feel free to browse the documentation online, download a copy to host yourself, or even help us out over on GitHub.
