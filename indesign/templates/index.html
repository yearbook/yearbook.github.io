InDesign API documentation

A Yearbook Machine project.

We at Yearbook Machine make extensive use of the InDesign API on a dailyb basis, but found the documentation system difficult to deal with. This is our solution to the problem. An attempt at making the InDesign API documentation more accessible and easier to use than the native Object Model Viewer.

Feel free to browse the documentation online, download a copy to host yourself, or even help us out over on GitHub.
